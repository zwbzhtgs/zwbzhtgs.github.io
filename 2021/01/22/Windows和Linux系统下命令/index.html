Windows和Linux下系统命令

发表于2021-01-22|更新于2021-01-22|棋魂
|阅读量:

Windows系统下命令

1、打开控制台

Windows+R/cmd回车

2、控制台操作

  • 切换盘符：C：
  • 显示所有文件和文件夹：dir
  • 查看选项菜单：dir /?
  • 清屏：cls
  • 终止：ctrl+c
  • 进入到某一个文件夹：cd xxx
  • 当前文件夹：cd.
  • 上一级文件夹：cd..
  • 返回盘符：cd/
  • 从某一盘符直接跳转其他盘符某文件夹时没有效果，但是跳转到该盘符时将直接跳转到该盘符的制定文件夹下
  • 创建文件夹：md a\b\a\b
  • 复制文件a到b文件夹（/-y是指同名覆盖时进行提示，/y则不提示，默认不提示）：copy /-y a b
  • 创建空文件使用：copy nul a.txt
  • 创建有部分内容的文件（文件内容为helloworld）：echo helloworld>c.txt
  • 移动文件a到b文件夹（/-y是指同名覆盖时进行提示，/y则不提示，默认不提示）：move /-y a b
  • 删除文件（删除某个文件，或者删除某目录下的所有文件[文件夹删不了]）：del a.txt 或 del myPic
  • 删除文件夹（默认只能删除文件夹，删除文件夹内所有内容需要加/s）：rd D:\myPic 或 rd /s mypic
  • 修改名字：ren oldname newname
  • 关机（-s关机-t关机倒计时）：shutdown -s -t 60
  • 关机（取消关机）：shutdown -a
  • 查看本机ip：ipconfig
  • 判断连接：ping ip地址（或直接填网址www.baidu.com）
  • 安装msi文件：msiexec /package “你的msi文件的路径”

3、注意事项

  • 在此命令体系中，文件夹之间的分割使用\斜杠
  • 命令使用**/斜杠**

Linux系统命令

1、呼出控制台

ctrl+alt+t

2、操作指令

  • 查看文件夹：ls

  • 查看包含隐藏文件的list（-a包含. 和..）：ls -a

  • 查看包含隐藏文件的list（-A不包含. 和..）：ls -A

  • 查看包含文件信息的list：ls -ls

  • 直接查看某文件的信息：ll a

  • 直接跳转到home文件夹：cd ~

  • 跳转回之前的文件夹：cd -

  • 回到上一级文件夹：cd ..

  • 回到上三级文件夹：cd ../../..

  • 显示当前路径：pwd

  • 创建文件夹：mkdir

  • 递归创建多级文件夹（里面不能有文件）：mkdir -p a/b/c

  • 递归删除多级文件夹（里面不能有文件）：rmdir -p a/b/c

  • 删除a文件：rm a

  • 删除a文件夹（仅删除空文件夹）：rmdir a

  • 删除a文件夹及其下的所有文件夹、文件（-r递归删除,-f表示强制删除,-v显示指令执行过程）：rm -rf a 或者 rm -r a 或者rm -r -f a

  • 删除时增加提示（-i互动模式，删除前进行指令提示）：rm -r -i a

  • 在b文件夹下创建a.txt文件：touch a.txt

  • 复制a.txt文件到b文件夹：cp a.txt b

  • 复制a.txt文件到上一级的b文件夹下，改名为c.txt：cp a.txt ../b/c.txt

  • 移动a.txt文件到c文件夹：mv a.txt a/b/c

  • 移动时显示文件移动过程：mv a/b/c/* x/y/z -v

  • 修改文件名：mv a.txt b.txt

  • 强制覆盖（-f）：mv -f x/y/z/abc a/b/c

  • 同名跳过（-n）：mv -n x/y/z/abc a/b/c

  • 查看环境变量配置信息：env

  • 设置环境变量（从/home开始）：WS=/home/zh/Desktop/x/y/z

  • 跳转到环境变量：cd $WS

  • 发布环境变量，使所有用户都可以使用：export ws

  • 创建压缩tar文件（c：压缩，v：显示打包过程，f：打包后的文件名）：tar -cvf a.tar a.txt

  • 多个文件联合打包：tar -cvf pack.tar abc ab

  • 解压到当前文件夹：tar -xvf pack.tar

  • 解压到a文件夹中（-C表示欲解压到的目录）：tar -xvf pack.tar -C a

  • 创建zip压缩文件（文件夹内的内容不会被添加）：zip pack.zip a.txt desk.tar

  • 创建zip压缩文件（使用-r将文件夹中的内容递归压缩，加v显示压缩过程）：zip -rv a.zip a

  • 指定创建zip压缩文件时压缩等级（1-9，耗时增加，压缩效果增强）：zip -rv9 d.zip a

  • 创建压缩文件后将原文件删除：zip -rvm k.zip a

  • 解压zip文件到当前文件夹：unzip d.zip

  • 解压zip文件到my文件夹中（-d指定文件解压位置）： unzip -d my d.zip

  • 创建gz格式压缩文件（只能压缩文件，不能压缩文件夹）：gzip -c k >d.gz

  • 将a文件夹中的文件都进行压缩（单个压缩）：gzip -r a

  • 将a文件夹中的压缩文件都进行解压：gunzip -r a

  • 解压某.gz格式压缩文件：gunzip k.gz

  • 修改root账户密码：sudo passwd root

  • 进入root账户：su root

  • 创建新用户：useradd hello

  • 创建新用户密码：passwd hello

  • 切换用户hello：su hello

  • cat命令：

    • 逐行显示文本内容：cat -n a.txt

    • 逐行显示文本内容及符号（比如下面显示出Tab符^I）：cat -A a.txt

    • 连接两个文件并生成总文件，内容连接：cat a.txt b.txt >c.txt

  • 分屏显示（b向前翻页，空格向后翻页，回车下一行，q退出）：more c.txt

  • 显示头部：head c.txt

  • 进入文件浏览，并分屏显示（“pgup”和“pgdn”翻页，键入“q”退出）：less c.txt

  • 下载文件：wget https://网址

3、Linux查看用户（只能root账户查看）

  1. 回到根盘：cd /

  2. 进入etc文件夹：etc

  3. 查看文件清单：ls

  4. 查看/编辑passwd文件：

    • 查看文件：cat passwd

    • 编辑文件：view passwd（需要使用:q指令退回）

  5. 展示内容：

    • zh：账号名
    • x：密码
    • 1000：uid，最多65535个，前500个系统使用
    • 1000：组id
    • zh,,,：描述
    • /home…：主目录

4、查看shadow影子文件

  1. 最前面的一长串：系统算法加密后密码
  2. 18570：创建时间距1970年1月1日的天数
  3. 0：表示密码随时可以修改，设置1时为1天最多可以修改1次，设置10为10天最多可以修改1次
  4. 99999：密码有效期时间，单位为天
  5. 7：距到期前7天进行提醒修改密码

5、创建用户并赋予权限

  1. -u：添加用户ID
  2. -g：设置组
  3. -G：设置附加组
  4. -d：建立home目录，一般在前面要加上-m
  5. -s：添加权限，/bin/bash为添加所有权限
  6. zh2：所设置的账户名

6、使用VIM编辑文档

  1. 下载vim：sudo apt-get install vim

  2. 使用vim：vim a.txt

  3. 编辑：在光标处按“i”可以在该位置进行插入操作。按“o”进行换行插入操作

  4. 退出：

    • “:q!”：不保存强制退出
    • “:wq”：保存并退出

感谢观看！
